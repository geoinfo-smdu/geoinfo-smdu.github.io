Bem-vindos à página do Github do GEOINFO!

O GEOINFO é uma das coordenadorias da Secretaria Municipal de Desenvolvimento Urbano da Prefeitura de São Paulo, Brasil.

"GEOINFO" é apenas um apelido, pois nosso nome oficial é Coordenação de Produção e Análise da Informação.

Infocidade

O Infocidade apresenta dados sobre o município de São Paulo em formato de tabelas, contendo dados brutos ou dados processados.

Abrir

Observa Sampa

O Observa Sampa permite consultar dados sobre o município de São Paulo de maneira interativa, contendo dados brutos e dados processados, assim como indicadores elaborados dentro do próprio departamento.

Abrir

GeoSampa

O GeoSampa é uma plataforma de visualização e acesso de dados georreferenciados relacionados ao município de São Paulo.

Abrir

Informes Urbanos

Os Informes Urbanos são uma publicação que traz análises sintéticas sobre vários temas de interesse para a cidade de São Paulo. São estudos elaborados sobre dados demográficos, sociais, econômicos, de uso do solo, sempre sob a ótica da dimensão territorial.
